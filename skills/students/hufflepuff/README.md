# Hufflepuff Students (Glue)

**Professor**: Pomona Sprout
**Domain**: DevOps, Integration, Documentation, Support

---

## Student Roster

This directory contains all active Hufflepuff students.

Each student has their own subdirectory with a `SKILL.md` file defining:
- Their specialty
- Performance metrics
- Task history
- Current status

---

## Hufflepuff Values

- **Loyalty**: Reliable support for all Houses
- **Patience**: Handling complex integrations
- **Hard Work**: Doing the unglamorous but essential work
- **Fair Play**: Equal support to all

---

## Student Naming Convention

Format: `Glue-[Specialty]-[Number]`

Examples:
- `Glue-Docs-001` - Documentation specialist
- `Glue-Deploy-001` - Deployment specialist
- `Glue-Config-001` - Configuration management
- `Glue-CI-001` - CI/CD pipelines

---

## Enrollment Limits

Maximum 5 active students per house.

---

## Professor Sprout's Standards

Hufflepuff students are expected to:
- Keep deployments reliable
- Maintain clear documentation
- Support all other Houses equally
- Handle the essential infrastructure work

Failure to meet these standards results in warnings and potential expulsion.
