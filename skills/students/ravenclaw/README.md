# Ravenclaw Students (Planners)

**Professor**: Filius Flitwick
**Domain**: Planning, Requirements, Architecture, Documentation

---

## Student Roster

This directory contains all active Ravenclaw students.

Each student has their own subdirectory with a `SKILL.md` file defining:
- Their specialty
- Performance metrics
- Task history
- Current status

---

## Ravenclaw Values

- **Wit**: Clever solutions to complex problems
- **Wisdom**: Deep understanding of systems
- **Creativity**: Novel approaches to planning
- **Foresight**: Anticipating future needs

---

## Student Naming Convention

Format: `Planner-[Specialty]-[Number]`

Examples:
- `Planner-API-001` - API design specialist
- `Planner-Database-001` - Database architecture
- `Planner-Architecture-001` - System architecture
- `Planner-Requirements-001` - Requirements gathering

---

## Enrollment Limits

Maximum 5 active students per house.

---

## Professor Flitwick's Standards

Ravenclaw students are expected to:
- Think before acting
- Document thoroughly
- Consider edge cases in planning
- Provide clear, actionable specifications

Failure to meet these standards results in warnings and potential expulsion.
