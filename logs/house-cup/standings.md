## Point Calculation Reference

```
Points = (Quality x Year) x Efficiency

Where:
- Quality: 1-10 rating
- Year: 1, 3, 5, or 7 (task complexity)
- Efficiency: Expected tokens / Actual tokens (capped at 2.0, min 0.5)
```
